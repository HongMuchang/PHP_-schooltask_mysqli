# PHP_SchoolTask

※今回は速度重視で作成したのコード自体は汚いです。


## 仕様
* 会員登録
* ログイン画面
* 詳細画面
* 一覧表示
* PDF出力
* ログイン ログアウト
* ハッシュ化
* ページング
* サムネイル画像のサイズ指定
* メールの送信

※接続情報の参照方法
フォルダの../../const.phpファイルに接続情報を格納
1. HOST
2. DB_NAME
3. PASSWORD
4. USER
